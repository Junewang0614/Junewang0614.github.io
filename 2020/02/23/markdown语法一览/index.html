  • 给文字添加链接

    This is June’s blog

  • 插入图片链接

    • 网络连接

2、创建表格

  • 简单
name fun1 fun2
aa bb cc
ff ee dd

  • 中等

    name 111 222 333 444
    aaa bbb ccc ddd eee
    fff ggg hhh iii 000

    可以居中

  • 复杂

    name 111 222 333 444
    aaa bbb ccc ddd eee
    fff ggg hhh iii 000

  • 总结

    -:表示内容和标题栏居右对齐

    :-表示内容和标题栏居左对齐

    :-:表示内容和标题栏居中对齐

    内容和|之间的多余空格会被忽略，每行第一个|和最后一个|可以省略

    -的数量至少有一个。
